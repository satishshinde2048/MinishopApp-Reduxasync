### `Tech Stack`

+ React js( React Router ),
     
+ Redux,       
     
+ TailwindCSS,  
     
+ React-icons,

+ Middleware

+ redux-thunk
     
     
### `Run Locally`

+ Clone the project:

      git clone https://github.com/satishshinde2048/MinishopApp-Reduxasync/.git


+ Navigate to the project directory:

      cd MinishopApp-Reduxasync
                                                                                                       
+ Install dependencies:                                                                                                    

       npm install 
                                                                                                                                                                                                                
+ Start the server:                                                                                                        

       npm start
